#kindle电子书分享及推送微信号
----

##系统功能##
- [x] **微信功能**
    - [x] 推送到邮箱
    - [x] 推送到kindle设备
- [x] **后台功能**
    - [x] 书籍上传
        - [x] 小于20MB的书籍直接保存到新浪云storage
        - [x] 大于20MB的书籍保存百度云盘链接及地址
    - [x] 书籍推送到邮箱或者kindle设备
    - [x] 书籍信息编辑或删除
    - [x] 注册人员管理
    - [x] 邀请码
    - [x] 日志管理

----

##说明##

- 采用新浪云的storage服务
- 发送邮件采用PHPMailer库
- 欢迎吐槽与指正
